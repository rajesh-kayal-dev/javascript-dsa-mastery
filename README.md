# JavaScript Data Structures and Algorithms

A comprehensive repository containing implementations of various data structures, algorithms, and technical interview preparation materials written in JavaScript.

## Contents

This repository includes solutions and practice files for:
- Basic algorithms (e.g., Factorials, Palindromes, Prime numbers)
- Number and string manipulation (e.g., Reversing strings, Summing digits, Vowel/Consonant checks)
- Advanced array operations and manipulations
- React-specific interview preparation
- Coding interview challenges from various platforms

## Usage

Each file contains isolated solutions or concepts. You can run them using Node.js directly from the terminal. For example:

```bash
node Factorial.js
```

## Structure

The repository is organized into general algorithm files in the root directory and specific focus areas (such as "React-Interview" or "OneWeekSprint") grouped within their respective subdirectories.
